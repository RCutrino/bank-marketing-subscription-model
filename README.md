# Bank Marketing Subscription Model
Machine learning project to predict customer subscription to a term deposit using the Bank Marketing Dataset. Includes data preprocessing, feature engineering, model comparison, and performance evaluation.

# Project overview
This project aims to predict whether a customer will subscribe to a term deposit based on historical marketing campaign data. The dataset comes from the Bank Marketing Dataset and includes demographic, financial, and campaign-related features. 

The goal is to build and evaluate machine learning models that can support marketing strategies by identifying high-probability customers, optimizing campaign efficiency, and reducing unnecessary contact costs.

# Objectives
- Perform data cleaning and preprocessing
- Conduct exploratory data analysis (EDA)
- Build classification models (Logistic Regression, Random Forest)
- Evaluate model performance using appropriate metrics (ROC-AUC, precision, recall)
- Interpret model results and extract business insights

# Business Value
By predicting customer subscription behavior, financial institutions can:
- Target the right customers more effectively
- Improve campaign conversion rates
- Reduce operational costs
